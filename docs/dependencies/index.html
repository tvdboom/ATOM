Python

As of the moment, ATOM supports Python 3.6, 3.7, 3.8 and 3.9.



Packages

ATOM is built on top of several existing Python libraries. The required packages are necessary for it's correct functioning. Additionally, you can install some optional packages to use machine learning estimators not provided by sklearn.

Required

  • numpy (>=1.17.2)
  • scipy (>=1.4.1)
  • pandas (>=1.0.3)
  • tqdm (>=4.35.0)
  • joblib (>=0.16.0)
  • typeguard (>=2.7.1)
  • tabulate (>=0.8.6)
  • scikit-learn (>=0.23.1)
  • scikit-optimize (>=0.7.4)
  • pandas-profiling (>=2.3.0)
  • category-encoders (>=2.1.0)
  • imbalanced-learn (>=0.5.0)
  • featuretools (>=0.17.0)
  • gplearn (>=0.4.1)
  • matplotlib (>=3.3.0)
  • seaborn (>=0.9.0)
  • shap (>=0.36.0)

Optional

  • xgboost (>=0.90)
  • lightgbm (>=2.3.0)
  • catboost (>=0.19.1)
